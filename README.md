# element-admin

## Project setup
```
npm install or cnpm install
```

### Compiles and hot-reloads for development
```
npm run serve，前端默认端口8080
```

### 启动后端服务
```
nodemon server,后端默认端口3001
```

### 启动第三方接口
```
requirements[Flask]
python app.py
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
